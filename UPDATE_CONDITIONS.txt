# Team Generator Update Conditions & Mandates

## 1. Zero Warning Policy
* The application MUST NOT display any warnings, popups, or error dialogs (e.g., CustomWarning, CustomInfo, messagebox.showerror).
* **SOLE EXCEPTION:** A popup is permitted ONLY when the user attempts to add a player who is already selected/present in the list.

## 2. Protected Players (Gotya & TUF)
* **PERMANENTLY REMOVED.** Do not re-implement any hidden admin keys, click counters, or point locks for specific players.

## 3. Visual Layout & Dimensions
* **Ratio:** Vertical/portrait ratio (~900x1000).
* **Background:** Solid dark color (e.g., #0f0f11). No semi-transparency inside the app.
* **Corners:** Window corners MUST be rounded (Corner Radius: 25).
* **Alignment:** Generator buttons must be perfectly aligned and evenly spaced.
* **Icons:** Use professional PNG icons for all buttons and section headers (Discord, Save, Load, Generate, Refresh, Trash).

## 4. Update Process
* When performing future updates, consult this file BEFORE writing any code to ensure these core mandates are never violated again.
